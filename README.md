# SF-Intel Studio

A Chrome extension that turns your browser into a full Salesforce IDE — edit Apex, LWC, and Aura components directly inside your org without switching to an external tool.

---

## Features

- **Apex / LWC / Aura editing** — open, edit, and deploy full bundles atomically
- **SOQL Runner** — multi-tab query editor with pagination, export (CSV / JSON / Excel), and saved queries
- **Diff against Org** — side-by-side diff of local vs org with per-hunk revert
- **Fetch Latest** — pull the current org version into any open tab
- **Schema Explorer** — interactive force-directed graph of object relationships
- **Log Inspector** — parse debug logs into Tree / Timeline / Errors / Limits views
- **Test Runner** — discover, run, and view coverage for Apex test classes
- **Automation Inspector** *(beta)* — trace the full trigger / flow / DML chain for a record save

---

## Installation

### Chrome Web Store
Search for **SF-Intel Studio** or install from the Chrome Web Store listing.

---

## Usage

1. Navigate to any Salesforce org in Chrome
2. Click the SF-Intel Studio icon in the toolbar to open the IDE
3. Browse Apex classes, LWC, or Aura components in the sidebar
4. Open a file, make changes, and click **Save & Deploy**
